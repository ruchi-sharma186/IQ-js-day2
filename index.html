Question 8: Explain Local Scope, Block Scope, Functional Scope and Scope Chain

Local Scope: Local scope means that a variable is accessible only inside the function where it is defined.
Block Scope: Block scope applies to variables declared with let or const inside a pair of curly braces {} like in if, for, etc.
Functional Scope:This means variables declared with var are limited to the entire function, not just inside the block.
Scope Chain: \The scope chain is the process JS uses to look for variables: it starts from the current scope and moves outward until it finds the variable.

Question 10: map and filter example

map: "creates a new array by applying a function to each element of the original array."
filter:creates a new array that contains only the elements that pass a certain condition (return true in the callback).

        const numbers = [1, 2, 3, 4, 5, 6];

// Using map to square each number
const squares = numbers.map(num => num * num);

// Using filter to get only even numbers
const evens = numbers.filter(num => num % 2 === 0);

console.log("Original array:", numbers);
console.log("Squares (map):", squares);
console.log("Even numbers (filter):", evens);

Question 11: Count 0's in an array

        const binaryArray = [0, 1, 0, 1, 0, 1, 1, 0, 0];

        let zeroCount = 0;
        
        for (let num of binaryArray) {
            if (num === 0) {
                zeroCount++;
            }
        }
        
        console.log("Array:", binaryArray);
        console.log("Number of 0's:", zeroCount);

Question 12: Count odd and even numbers in an array

        const numbers = [10, 23, 44, 55, 66, 71, 80];

        let oddCount = 0;
        let evenCount = 0;
        
        for (let num of numbers) {
            if (num % 2 === 0) {
                evenCount++;
            } else {
                oddCount++;
            }
        }
        
        console.log("Array:", numbers);
        console.log("Even numbers:", evenCount);
        console.log("Odd numbers:", oddCount);
